# Community Radio Hub

Discover and listen to Australian community radio stations.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Browse 1,500+ Australian radio stations from the Radio Browser API
- Search, filter by region, genre, and First Nations stations
- Live streaming with persistent player bar

## Refresh Station Data

```bash
node scripts/fetch-radio-browser-stations.js
```
